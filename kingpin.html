Key Features

Login & Team Code Coach login and a 4-digit team entry flow.
Dashboard Top scorers, awards, match center, and settings.
Match Editing Score input, match browsing, and edit mode.
Player Stats Per-player averages and team breakdowns.
Awards Create and manage awards from the dashboard.
Export & Settings Spreadsheet export and account settings tools.

Pages In The Build

  • login.html handles coach login and team-code entry.
  • dashboard.html surfaces the team overview and match center.
  • statInput.html is the scoring/input workflow.
  • matchView.html shows the per-match breakdown and edit mode.
  • teamScores.html, plyrScores.html, and player stat pages show team and player analytics.
  • settings.html and signUp.html manage account setup and preferences.

How It Was Designed

  • The backend docs show a Firestore-first structure with per-user and per-team collections.
  • Middleware strips passwords from GET routes and validates passwords for write routes.
  • The UI is split into focused screens for login, dashboard, scoring, stats, and settings.
  • The workflow is built for coaches: log in, select a team, enter scores, review stats, and export when needed.

Navigation

Dashboard sections act like a control center, while the detailed views isolate one task at a time.

Data Model

Accounts, teams, players, matches, and awards are stored in Firestore with predictable schemas.
